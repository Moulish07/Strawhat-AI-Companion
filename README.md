# LUFFY - AI Voice Assistant

A truly modern, Generative AI desktop voice assistant built from scratch in Python. Featuring a sleek cyberpunk Graphical User Interface (customtkinter) and integrated with Google's Gemini LLM for infinite conversational intelligence.

## Features
- 🎙 **Speech Recognition & Text-to-Speech:** Continuously listens for the wake word (`Luffy`).
- 🧠 **Google Gemini 2.5 Brain:** Infinite generative intelligence. Can organically answer any factual, contextual, or conversational prompt.
- 🤖 **Local OS Control:** Physically opens desktop applications (Notepad, Calculator) upon command.
- 🎵 **Browser Automation:** Able to automatically scrape, open, and play YouTube music videos dynamically.
- 💬 **WhatsApp Automation:** Programmed with an internal phonebook to instantly send dynamic messages through WhatsApp Web.
- ⛅ **Real-Time Web Scraping:** Seamlessly accesses live internet data (Weather, Time) without complex API limitations.
- 💻 **Cyberpunk GUI:** Built using `customtkinter`, catching all system outputs gracefully inside a dark-mode graphical wrapper.

## Installation & Run
1. Clone the repository.
2. Initialize virtual environment: `python -m venv venv`
3. Activate it and install dependencies: `pip install -r requirements.txt`
4. Add your free Gemini API key to `jarvis.py`.
5. Run the UI: `python gui.py`
